# UI/UX

# Prompt Thiết Kế UI/UX Hệ Thống AI Learning Platform

## Mục tiêu

Thiết kế giao diện UI/UX hiện đại cho một hệ thống học tập tích hợp AI tên là **EduAI**.

Phong cách thiết kế:

- Futuristic
- Dark theme
- Neon gradient
- Glassmorphism nhẹ
- Dashboard chuyên nghiệp
- Hiện đại giống sản phẩm startup AI quốc tế
- Tập trung trải nghiệm học tập cá nhân hóa bằng AI

---

# Tổng thể thiết kế

## Tone màu

Sử dụng tone:

- Dark navy
- Purple neon
- Blue electric
- Gradient tím xanh
- Highlight glow effect

Ví dụ:

- Background: #070B1A
- Card: #0F172A
- Border glow: rgba(139,92,246,0.4)
- Primary gradient: linear-gradient(135deg,#6D28D9,#9333EA,#3B82F6)

Sidebar vertical hiện đại.

Menu gồm:

1. Tổng quan
2. Lộ trình học tập
3. Kỹ năng
4. Bài kiểm tra
5. AI Mentor
6. Cơ hội nghề nghiệp
7. Khóa học & tài liệu
8. Dự án thực hành
9. Thống kê & báo cáo
10. Cộng đồng
11. Cài đặt

Sidebar có:

- Icon Lucide/Phosphor
- Hover glow effect
- Active item gradient tím
- Animation mượt
